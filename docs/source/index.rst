PatentsView-Evaluation's documentation
======================================

.. currentmodule:: pv_evaluation

PatentsView-Evaluation is a Python package for the evaluation and benchmarking of PatentsView disambiguation algorithms.

To get started, please refer to the :doc:`README <readme>`, the :doc:`examples <examples>` page, or the :doc:`API documentation <pv_evaluation>`.
Use the project's `Discussion <https://github.com/PatentsView/PatentsView-Evaluation/discussions>`_ page to ask questions and the `Issues <https://github.com/PatentsView/PatentsView-Evaluation/issues>`_ page to request features and report bugs.
